# Epoxy Resin Volume Calculator
Add the script to generate the volume calculator, preferably on footer section inside the <code>&lt;body&gt;</code> tag:
(external js file)
```
<script type="text/javascript" src="./index.js"></script>
```
or (inline js)
